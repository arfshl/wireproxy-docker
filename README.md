#### Unnofficial docker image for [wireproxy](https://github.com/windtf/wireproxy)

#### Tag list
- Debian Base

      docker pull arfshl/wireproxy:latest
      docker pull arfshl/wireproxy:debian


- Alpine Base

      docker pull arfshl/wireproxy:alpine

Supported arch: `amd64` `arm64` `arm/v7` `ppc64le` `s390x` `riscv64`

-  Legacy architecture, based on `debian:bookworm` for `386` and `mips64le` and `debian:trixie` for `arm/v5`

       docker pull arfshl/wireproxy:debian-386
       docker pull arfshl/wireproxy:debian-mips64le
       docker pull arfshl/wireproxy:debian-armv5
       
Sources: https://github.com/arfshl/wireproxy-docker
